# Simple calculator program

## Description 
This program is a training on how to write make files as a part of the intensive training program provided by iti. This calculator applies the basic arthmatic operations + - * / , you should first enter the operations then the two opearands separated by a space 

## How to build and run 
open your terminal in the directory of the project and then execute the command make then you will find the app binary in the /build directory. 

```
make 
./build/app 
```
then interact with the program as following : 
```
Enter an operator (+, -, *, /): +
Enter two operands: 23 21
23.0 + 21.0 = 44.0
```
